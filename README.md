# git-whiteboard

Website to create and interact with a graph drawn with HTML canvas.

## Setup

```bash
yarn

yarn dev
# or start the server and open the app in a new browser tab
yarn dev -- --open
```

## Building

To create a production version:

```bash
yarn build
```
